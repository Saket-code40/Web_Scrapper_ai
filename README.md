# AI Web Scraper

An AI-powered web scraping application built using Python, Selenium, BeautifulSoup, Streamlit, and LLMs.

## Features

* Scrape website content from any URL
* Extract and clean DOM content
* AI-powered information extraction
* Streamlit web interface
* Chrome browser automation using Selenium

## Tech Stack

* Python
* Selenium
* BeautifulSoup
* Streamlit
* LangChain
* Ollama

## Installation

```bash
git clone https://github.com/Saket-code40/Web_Scrapper_ai.git

cd Web_Scrapper_ai

pip install -r requirements.txt

streamlit run main.py
```
